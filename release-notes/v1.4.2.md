# GreenFrog v1.4.2

## Bug Fixes

- **Login auto-fill**: Opening the web UI from localhost now automatically pre-fills the access token — no more manual copy-paste from config.yaml. Just click "进入平台".
- **Multi-turn chat**: Fixed 400 error when sending a second message after the AI has already replied (assistant history was encoded with wrong content type for the Responses API).

## Upgrade

Re-run the installer from the new package. Your data directory and config are preserved.
